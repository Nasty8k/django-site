# Requirements

* Python (3.6, 3.7, 3.8)
* Django (3.0.4)

# Example
* Linux:


$ python3 -m pip install --upgrade pip
$ sudo apt install python3-django

* Windows:

pip install Django

Start project in  Windows PowerShell
 
 
   cd django-site
   python ./manage.py runserver
